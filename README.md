# Gestion-Aeropuerto

Objetivo del proyecto
El objetivo principal del proyecto es desarrollar un sistema de gestión para un aeropuerto, que permita organizar, almacenar y consultar de manera eficiente toda la información relacionada con:

Compañías aéreas que operan en el aeropuerto

Aviones asociados a cada compañía

Vuelos programados (origen, destino, horario, estado)

Empleados de las compañías (pilotos, azafatas, técnicos, administrativos)

Puertas de embarque utilizadas en las operaciones

Pasajeros registrados en los vuelos

Reservas de asientos realizadas por los pasajeros

Este sistema debe facilitar tanto la gestión interna del aeropuerto como mejorar la experiencia de usuario para los pasajeros y empleados, mediante la organización clara de todos los elementos logísticos y operativos.
